Quick Start

1. Install
Windows: run obs-vdoninja-windows-x64-setup.exe.
Fallback: extract zip, then run install.cmd.
2. Configure OBS
Open Settings -> Stream, choose VDO.Ninja, and set Stream ID.
3. Go live
Start streaming, then open your viewer link using ?view=StreamID.
Password-enabled streams append a hash internally. Viewer links and room solo links still use the unhashed Stream ID.
