OBS VDO.Ninja Plugin

Native OBS plugin for VDO.Ninja publishing and ingest over WebRTC.

Capabilities

  • Publish OBS output to VDO.Ninja using push links.
  • Receive VDO.Ninja streams as OBS sources using view links.
  • Support multi-viewer sessions from one publisher.
  • Data-channel parsing for control metadata and WHEP URL hints.

Install

See the repository README and Releases for binaries and platform instructions.
